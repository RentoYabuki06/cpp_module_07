# cpp_module_07

## Overview
This module covers the fundamental concepts of template programming in C++. You will learn how to implement generic data structures and functions using C++ templates.

## Exercises

### ex00: Function Templates
- Implementation of function templates for minimum, maximum, and swap operations
- Creating template functions that can work with any data type

### ex01: Iterator Class Template
- Manipulating container elements using iterators
- Implementing template algorithms

### ex02: Array Class Template
- Implementation of a class template representing a fixed-size array
- Implementation of safe access methods including exception handling

## Compilation
Each exercise can be compiled using the following commands:
```bash
cd exXX
make
```

## Learning Objectives
- Understanding the basic concepts of template programming
- Creating type-independent generic code
- Understanding template specialization and implementation details

## Evaluation Criteria
- Code accuracy and efficiency
- Proper use of template features
- Error handling and testing of boundary conditions
